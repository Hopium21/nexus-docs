## The Network

The Nexus Network is a massively-parallelized proof network for executing and proving the Nexus zkVM. It is still under active development but you can [contribute compute to the network](/network). 
See the [Network](/network) page for more information.

**Important:** *The Nexus network is currently in devnet. It is important to note that you cannot earn Nexus points. Instead, devnet allows developers to experiment and build with the network. Stay tuned for updates regarding future testnets.*
## The zkVM

The [Nexus zkVM](/zkvm) can prove any computation. It is a general purpose VM bringing the best advances in ZK to the widest possible audience.
